核心功能（逐项说明）
- 章节生成：根据大纲、前文与设定自动生成高质量章节文本，支持多种风格与长度控制。
- 文章精修：智能润色与风格套装，快速修正语病、提升文笔、统一叙事风格。
- 大纲管理：可视化大纲编辑与版本回滚，支持场景/章节拖拽重组。
- 角色管理（角色卡）：人物卡片化管理，包含性格、口吻、禁忌与决策规则，保证角色言行一致。
- 伏笔系统：自动提取、标记与提醒待回收伏笔，避免伏笔丢失并提高故事张力。
- 章节分析：自动抽取章节摘要、情节点、情感弧线与冲突点，便于快速把控节奏。
- 世界观设定：集中管理世界规则、背景资料与专有名词，生成时自动保持一致性。
- 多格式导入：支持 Markdown、Word、HTML、TXT、PDF 等，快速导入既有作品与素材包。
- 混合分析素材导入：本地规则 + 低成本AI混合分析，成本低、准确率高，适合批量素材处理。
- 多AI角色讨论系统：可创建多个AI角色并以人设讨论情节，模拟人物互动与多方信息流转。
- 向量记忆系统（本地向量库）：为章节、人物与世界观建立向量记忆，实现语义检索与长上下文保持。
- 上下文筛选缓存：智能缓存上下文筛选结果，减少重复AI调用，加速章节生成。
- 素材库与去重：管理本地素材包，自动去重与标签化，快速复用高质量素材。
- 向量检查员：可视化查看/删除向量条目，保障续写质量与隐私控制。
- API与模型兼容：支持接入多种国产/第三方AI API（例如通义千问、智谱等），并可配置低成本模型以节约费用。
- 激活与权限管理：支持激活码、免费版/专业版区分、功能隔离与安全保护。

技术与竞争优势
- 混合引擎：本地规则快速处理 + 经济模型AI做深度识别，成本可控且对中文场景优化。
- 多步骤推理（CoT）与分层生成：先推理情感弧线与情节点，再分场景生成，逻辑更严密。
- 角色卡 + 短期/长期记忆：保证人物在长篇中的一致性与成长轨迹。
- 向量化+RAG策略：向量记忆用于检索上下文，结合生成模型提供事实一致的续写。
- 可配置策略：缓存策略、向量阈值、Token预算等可调，适配不同规模的写作任务与成本侧重点。
